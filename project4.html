Project Overview

This Robo Car is a compact, Bluetooth-controlled robot built around the AT89S52 (8052) microcontroller. Control commands are sent from a custom Android app Serial Bluetooth Terminal to the HC-05 Bluetooth module connected to the 8051 UART. The received commands are parsed by the firmware and mapped to motor control functions (forward, backward, left, right, stop).

The vehicle uses a 2-wheel differential drive (two DC geared motors) and a caster wheel for balance. Motor driving is handled by an L298N H-bridge and the microcontroller generates PWM to control motor speed smoothly. Power comes from a 12V battery pack (4 √ó 18650 cells) with a 7805 regulator for logic supply,or directly from motor driver.

Challenges & Solutions

Motor jerk / abrupt start: fixed using PWM ramp-up for smoother acceleration.

Voltage dips under load: separated motor & logic supply, added decoupling caps and a 7805 regulator for MCU.

Bluetooth command mismatches: used fixed single-character ASCII commands and added simple input validation in firmware.
